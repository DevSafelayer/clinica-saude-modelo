## 🎯 Sobre o projeto

Site institucional completo para clínicas de saúde integrativa. Desenvolvido com foco em **conversão de pacientes**, **design profissional** e **performance**.

Este modelo foi criado para demonstrar como um site moderno pode ajudar clínicas, consultórios e profissionais de saúde a:

- Atrair novos pacientes pelo Google
- Transmitir credibilidade e confiança
- Facilitar o agendamento via WhatsApp ou formulário
- Apresentar a equipe e os serviços de forma clara

---

## ✨ Funcionalidades

- ✅ **Design responsivo** — funciona perfeitamente em celular, tablet e desktop
- ✅ **Hero com animações** — apresentação impactante com estatísticas e depoimento
- ✅ **Seção de serviços** — cards com hover effect para cada especialidade
- ✅ **Sobre a clínica** — layout assimétrico com citação destacada
- ✅ **Equipe** — cards dos profissionais com especialidades e registro
- ✅ **Processo de atendimento** — 4 etapas visuais e claras
- ✅ **Formulário de agendamento** — campos de nome, WhatsApp e especialidade
- ✅ **Botão WhatsApp** — contato direto com um clique
- ✅ **Otimizado para SEO** — meta tags, descrição e estrutura semântica
- ✅ **Animações suaves** — reveal on scroll com IntersectionObserver

## 🚀 Como usar

1. Clone o repositório:
```bash
git clone https://github.com/devsafelayer/clinica-saude-modelo.git
```

2. Abra o `index.html` no navegador — sem instalar nada.

3. Para personalizar, edite diretamente no arquivo:
   - **Nome da clínica** — busque por "Espaço Viver"
   - **Especialidades** — seção `#servicos`
   - **Equipe** — seção `#equipe`
   - **Contato** — WhatsApp, e-mail e endereço na seção `#contato`
   - **Cores** — variáveis CSS no `:root`
